About Web Page Reader


Finally, a web page reader that delivers a smooth experience.


Web Page Reader makes no excuses. It's great for students, the vision impaired, people learning a new language or just people wanting to learn more, faster, and comprehend what they are reading better.

This extension reads web pages with text to speech (TTS) and highlights one sentence at a time. It doesn't require you to select any text first. By default it reads the whole page.

If you do select any text, it starts reading at your selection, but continues on from there until you stop it, or it reaches the end of the page. You can go forwards and backwards, skipping to the next or previous sentence.

You can control the voice, speaking rate and volume from the extension control panel. It loads all the voices on your machine.

You can also control the reader entirely from the keyboard. See the keyboard image for key assignments.
